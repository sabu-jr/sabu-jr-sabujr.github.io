Lost in the Museum


This project was completed as a coursework during my MSc. The aim of this project is to showcase my understanding and expertise of Game design principles.

Part one of this document is a propsal for a cultural heritage game : Lost in the Museum. This section describes describes the core gameplay, design drivers, appealing features, and which game design patterns are used and how.

Second part is a proposal for the game. The proposal have the form of a pitching session for a fund of a local council.

Next part describe the design issues I have encountered on this project and lessons I have learned during the period, referring to game design theory. This section also include a reflection on my own design work and thinking.

The final part is the game design document for the proposed game. This document describes an overview of the game explaining the consept, style and features. This document also describes the game world and narrative of this game. Other important elements like the formal game elements, core loop, and mechanics are also described in this design document.
